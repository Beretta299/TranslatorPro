How It Works

Three steps. A few seconds. Any language.

1

Press the shortcut

Hit Cmd+Shift+X on Mac or Ctrl+Shift+X on Windows to activate capture mode.

2

Select the area

Drag to select any region of your screen that contains text you want to translate.

3

Read the translation

A floating popup shows the original text and translation. Copy it, save words, or dismiss.
